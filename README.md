# Portfolio Clone

This is a clone of the Isabel Bakhshi portfolio website homepage.

## Files Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - JavaScript for interactivity
- `images/` - Directory for project images

## Images Needed

To complete the portfolio, you'll need to add the following images to the `images/` directory:

1. `tap-to-pay.jpg` - Tap to Pay on iPhone project image
2. `outage-mode.jpg` - Outage Mode project image
3. `square-one.jpg` - Square One project image
4. `cash-app-square.jpg` - Cash App Square project image
5. `ai-insights.jpg` - AI-Generated Business Insights project image
6. `ai-library.jpg` - AI Item Library Generator project image

You can download these images from https://www.isabelbakhshi.com/ or use your own images.

## How to Use

1. Add the project images to the `images/` directory
2. Open `index.html` in a web browser
3. The page is fully responsive and will work on desktop, tablet, and mobile devices

## Features

- Clean, minimalist design matching the original
- Responsive layout for all screen sizes
- Smooth scrolling navigation
- Hover effects on work items
- Professional typography using Inter font

## Customization

You can customize:
- Colors in `styles.css`
- Content in `index.html`
- Interactive features in `script.js`
